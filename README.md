# S3_staticwebsite_hosting
Host a website without any server using Amazon S3.
This project demonstrates how to host a static website using Amazon S3.

## Technologies Used
- Amazon S3
- HTML
- CSS

## Steps
1. Create S3 bucket
2. Enable static website hosting
3. Upload website files
4. Configure bucket policy
5. Access website through S3 endpoint

## Outcome
Serverless static website deployed using Amazon S3.
## Live Demo
http://chandan_portfolio.s3-website-us-east-1.amazonaws.com/

This project was deployed using Amazon S3 Static Website Hosting in an AWS sandbox environment.

Note: The sandbox environment may reset, so the live URL may not always be active.
